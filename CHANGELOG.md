# Changelog

Tous les changements notables de ce projet seront documentés dans ce fichier.

## [1.0.0] - 2024-12-14

### Ajouté
- Site web complet pour André Bellemare
- Galerie de toiles avec 6 catégories
- Système de recherche et filtres
- Modal d'achat avec envoi d'email
- Mode sombre/clair
- Lightbox pour les images
- Zoom au survol sur les fiches produit
- Partage social (Facebook, Pinterest)
- FAQ et témoignages gérables
- Administration Django personnalisée (`/alexandre/`)
- SEO optimisé (JSON-LD, sitemap.xml, robots.txt)
- Docker Compose avec PostgreSQL
- Makefile pour faciliter les commandes
- Dev Container pour VS Code
- Cloudflare Tunnel pour l'exposition publique
- Script de seed avec données de démo
- Support des images depuis `/tmp` ou génération automatique

### Configuration
- Variables d'environnement pour email Gmail
- Volumes Docker pour données persistantes
- Compression d'images avec sorl-thumbnail
- Tailwind CSS pour le design
