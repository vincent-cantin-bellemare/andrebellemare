# Guide de contribution

Merci de votre intérêt pour contribuer au site d'André Bellemare !

## Développement local

1. Clonez le dépôt
2. Créez votre `.env` à partir de `env.example`
3. Lancez `make setup` pour la configuration initiale
4. Créez un superutilisateur avec `make createsuperuser`

## Standards de code

- **Python** : Suivez PEP 8
- **Django** : Respectez les conventions Django
- **Templates** : Utilisez Tailwind CSS pour le styling
- **Commentaires** : En anglais dans le code, documentation en français

## Structure des commits

Utilisez des messages de commit clairs :
- `feat:` Nouvelle fonctionnalité
- `fix:` Correction de bug
- `docs:` Documentation
- `style:` Formatage
- `refactor:` Refactorisation
- `test:` Tests

## Tests

Avant de soumettre une PR, assurez-vous que :
- Le code fonctionne localement
- Les migrations sont à jour
- Aucune erreur de linting

## Questions ?

Contactez l'administrateur du projet.
